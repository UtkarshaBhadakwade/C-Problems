# C-Problems
Questions related to C 
#VARIABLE
1. Declare an `int`, a `float`, a `char`, and a `double` variable. Initialize each with a value and print all of them using `printf`.

2. Write a program that takes two integers as input (using `scanf`) and prints their sum, difference, product, and quotient.

3. Swap the values of two integer variables **without using a third variable**.

4. Write a program to find the size (in bytes) of `int`, `float`, `char`, and `double` using the `sizeof()` operator.

5. Take a temperature in Celsius as input and convert it to Fahrenheit using the formula: `F = (C * 9/5) + 32`. (Watch out for integer division here.)
